# Requirements

- docker (cli or desktop)

# Quick start 

```
git clone https://github.com/PurpleFlood/tor-proxy-docker-run && sh tor-proxy-docker-run/docker-run.sh
```

# Uninstall

```
sh uninstall.sh
```

# Exposed Ports

    9050: Tor proxy (SOCKS5)
    9051: Tor control port
    8118: Privoxy (HTTP Proxy)
